Engine Calculator

Github Source

Some incorporated technologies

  • React
  • HTML
  • CSS
  • JavaScript

This is a simple React app that I created to help me calculate the cubic inch displacement of an engine based off of number of cylinders, bore diameter, and stroke length. I only spent about an hour working on this, but I plan to add more features. I am also planning to add a backend to this project to store the data in a database and allow users to save their engines, or get common engine sizes to convert. I also plan to make this a PWA so that it can be used offline.

Check it out on Github for the latest info.
